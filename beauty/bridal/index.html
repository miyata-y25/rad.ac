    コース

※2年次からの専門コース専攻になります

ブライダルコースは、結婚式という特別な一日を、 「人」「空間」「演出」まで含めて支える仕事を目指すコースです。
プランニング、接客、衣装、美容の基礎知識を学び、 お客様の想いをかたちにするための提案力と現場対応力を身につけます。
ブライダル業界に限らず、 接客・サービス・空間演出など、人と深く関わる仕事につながる力を育てます。

Class Class 主な授業内容

ブライダルの仕事に必要な、実務に直結する技術と知識を総合的に学びます。

  • 現場実習

    （結婚式場・ドレスサロン）

    結婚式の準備から当日まで、リアルな現場を体験する実習です。
    結婚式場やドレスサロンにて、現場の流れ・役割・チームワークを学び、即戦力として求められる実践力を身につけます。

  • ブライダル知識

    （基礎・実務理解）

    ブライダル業界で働くために必要な基礎知識を体系的に学びます。
    ブライダルの歴史やしきたり、マナー、業務ルールまでを理解し、仕事に直結する知識を身につけます。あわせて、在学中に資格取得を目指します。

  • 提案・プランニング

    （実務力）

    お客様の想いをかたちにする、提案力・プランニング力を養います。
    ヒアリングから企画立案まで、ブライダル実務の中核となるスキルを演習を通して実践的に学びます。

  • ドレス・和装

    （衣装・コーディネート）

    ウェディングドレスや婚礼和装の基礎知識と着付けを学びます。
    衣装の特徴やコーディネート、実際の着せ方までを理解し、現場で役立つ衣装知識を身につけます。

  • 感性向上ワーク

    （イメージ力・表現力）

    お客様のイメージをくみ取るための感性を磨く授業です。
    多様な体験やワークを通して、想像力・表現力・美的感覚を高め、提案の幅を広げていきます。

  • ブライダルヘアメイク

    （美容基礎）

    新婦様のヘアメイクに必要な基礎技術を学びます。
    ブライダル美容の考え方を理解し、ヘアメイクの基礎知識と技術を身につけることで、現場理解を深めます。

  • フラワー

    （装花・ブーケ）

    ブーケや会場装花など、ブライダルフラワーの基礎を学びます。
    フラワーの役割や表現方法を理解し、空間演出に欠かせない要素としての装花知識と基礎技術を習得します。
