※2年次からの専門コース専攻になります

エステコースは、肌・身体・心に向き合い、お客様の「美と 健康」を支える仕事を目指すコースです。
フェイシャル・ボディ・リラクゼーションなどの施術技術に加え、美容理論や接客、サロンワークまでを学び、現場で信頼されるエステティシャンを目指します。
エステサロンはもちろん、美容業界全体で活かせるカウンセリング力・提案力・継続的な関係構築力を身につけます。

Class Class 主な授業内容

エステの仕事に必要な、実務に直結する技術と知識を総合的に学びます。

  • フェイシャル技術

    （肌ケア・施術力）

    フェイシャルエステに必要な施術技術を、基礎から応用まで学びます。
    フェイシャルマッサージや美容機器の扱い方を習得し、肌状態に合わせた施術ができる力を身につけます。

  • ボディトリートメント

    （全身施術）

    全身のボディマッサージ技術を学び、施術力を高めます。
    手技を中心に、エステティシャンとして求められる安定した施術スキルと身体の使い方を習得します。

  • リラクゼーション

    （多様な施術技術）

    世界のリラクゼーション技術を学び、施術の幅を広げます。
    専門学校では珍しい内容として、さまざまなリラクゼーション手法を取り入れ、お客様の目的や状態に合わせた対応力を養います。

  • 美容理論

    （知識・安全理解）

    施術の土台となる美容理論を体系的に学びます。
    皮膚学・解剖学など、エステを行う上で欠かせない知識を理解し、安全で効果的な施術につなげます。

  • 食育

    （インナービューティー）

    内面から美を考えるための知識を学びます。
    栄養や食事と美容の関係性を理解し、外側だけでなく内側からのアプローチができる視点を身につけます。

  • サロンワーク

    （接客・現場対応）

    エステサロンでの一連の流れを、実践的に学びます。
    お出迎えからカウンセリング、施術、お見送りまでを想定し、お客様第一の接客姿勢と対応力を身につけます。

  • 脱毛

    （施術知識・安全管理）

    エステサロンで主流となっている光脱毛について学びます。
    施術に必要な知識と技術、安全管理を理解し、正確で安心感のある対応ができる力を養います。
